# 🛡️ Fake Payment Screenshot Detector

An AI-powered system that detects **fake, edited, and duplicate payment screenshots** in real time using a camera. The project was developed to help canteens, shops, and small businesses prevent payment fraud during online transactions.

---

## 📌 Problem

Many small businesses and university canteens receive online payments through mobile banking apps. Some customers misuse this by showing:

* Fake payment screenshots
* Edited payment screenshots
* The same screenshot multiple times

This can lead to financial losses because sellers often verify payments manually.

---

## 💡 Solution

This project uses a camera and AI-based image analysis to verify payment screenshots instantly.

The system scans the screenshot and determines whether it is:

* ✅ Real
* ❌ Fake or edited
* 🔁 Duplicate (already used before)

This helps businesses verify payments quickly and securely.

## 🚀 How It Works

1. Open the application.
2. Show the payment screenshot to the camera.
3. The system analyzes the screenshot.
4. The result is displayed as:

   * Real Screenshot
   * Fake Screenshot
   * Duplicate Screenshot
